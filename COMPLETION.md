## How to Use

### Local Testing
1. Open `index.html` in a web browser
2. All features work instantly
3. No installation needed

### Deployment
1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Site deploys automatically
4. Access at: `https://<username>.github.io/Extra-Credit-Physic`

### Adding More Content
Edit these files to add more:
- **Notes:** Update `physicsData.notes` in `data.js`
- **Flashcards:** Add to `physicsData.flashcards` in `data.js`
- **Quiz Questions:** Add to `physicsData.quiz` in `data.js`
- **Problems:** Add to `physicsData.problems` in `data.js`

## Support & Troubleshooting

### If Quiz Timer Doesn't Work
- Ensure JavaScript is enabled
- Try a different browser
- Check browser console (F12)

### If Progress Isn't Saving
- Check if localStorage is enabled
- Try in private/incognito window
- Clear browser cache

### If Site Won't Load
- Ensure all files are in same directory
- Check file paths in HTML are correct
- Try refreshing the page

### For Deployment Issues
- See `DEPLOYMENT.md`
- Verify repository is public
- Check GitHub Pages settings
- Wait 2-3 minutes for deployment

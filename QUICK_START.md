# 🚀 Quick Start Guide

## 30 Second Setup

1. **Open the website:**
   - Double-click `index.html`
   - Or right-click → Open with → Your browser

2. **Start studying:**
   - Click on any section in the menu
   - All features work immediately!

## What You Get

### 📚 Notes
- Click topic → read content
- Progress auto-saves

### 🃏 Flashcards  
- Click card to flip
- Rate difficulty with buttons
- Navigate with arrows

### ❓ Quiz
- Select difficulty level
- Timer starts automatically ⏱️
- See score & time at end

### 📝 Problems
- Select category
- Click "Show Solution"
- Step-by-step answers

### 📊 Progress
- All stats auto-tracked
- Visual progress bars
- Reset anytime

## Deploy to GitHub Pages (2 minutes)

1. Go to your GitHub repository
2. Settings → Pages
3. Select "main" branch
4. Click Save
5. Wait 1-2 minutes
6. Visit: `https://<your-username>.github.io/Extra-Credit-Physic`

## Troubleshooting

**Quiz timer not showing?**
- Refresh the page
- Try a different browser

**Progress not saving?**
- Make sure localStorage is enabled
- Try incognito window

**Site not loading online?**
- See DEPLOYMENT.md
- Make sure repo is public
- Wait 2-3 minutes for deploy

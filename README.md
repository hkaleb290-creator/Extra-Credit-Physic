# Physics Study Hub 🔬

A comprehensive interactive physics study website with notes, flashcards, quizzes, and problem solutions.

## Features ✨

- **📚 Study Notes** - Organized by topic (Kinematics, Forces, Energy, Momentum, Waves)
- **🃏 Interactive Flashcards** - 10+ flashcards to test your knowledge
- **❓ Quiz Mode** - Multiple choice quizzes with difficulty levels and timer
- **📝 Practice Problems** - 16 solved problems across all topics
- **📊 Progress Tracking** - Automatically saves your learning progress
- **⏱️ Quiz Timer** - Track how long quizzes take
- **🎯 Difficulty Levels** - Filter quizzes by easy, medium, or hard

## How to Use

1. Open `index.html` in a web browser
2. Navigate between sections using the menu
3. Your progress is automatically saved in your browser

### Notes
- Click on different topics to review key concepts
- Each topic includes definitions and important formulas

### Flashcards
- Click a card to flip between question and answer
- Rate difficulty to track weak areas
- Navigate through all cards at your own pace

### Quiz
- Select a difficulty level (All, Easy, Medium, Hard)
- Answer multiple choice questions
- Timer shows how long the quiz takes
- See your score and time at the end

### Problems
- View practice problems with step-by-step solutions
- Click "Show Solution" to reveal answers
- Problems organized by topic

### Progress
- Track notes reviewed, cards studied, quiz scores, and problems solved
- Visual progress bars show your learning journey
- Reset progress if needed

## Topics Covered

1. **Kinematics & Motion** - Displacement, velocity, acceleration, equations of motion
2. **Forces & Newton's Laws** - Net forces, friction, tension, inclined planes
3. **Energy & Work** - Kinetic energy, potential energy, work, power
4. **Momentum & Collisions** - Momentum conservation, impulse, elastic/inelastic collisions
5. **Waves & Sound** - Wave properties, wavelength, frequency, Doppler effect

## Technical Details

- Pure HTML, CSS, and JavaScript (no dependencies)
- Data stored in browser's localStorage
- Fully responsive design
- No internet connection required after initial load

## GitHub Pages Deployment

This site is ready to deploy to GitHub Pages:

1. Push to your GitHub repository
2. Go to Settings → Pages
3. Select "Deploy from a branch"
4. Choose the main branch as source
5. Your site will be live at `https://<username>.github.io/<repo-name>`

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)

# Physics 1 Study Guide ⚛️

A static study website built for Physics 1 (extra credit). Open `index.html` in any browser — no build step required.

## Features

| Section | Description |
|---|---|
| **Core Topics** | Expandable accordion cards covering Kinematics, Newton's Laws, Work/Energy/Power, Momentum, Circular Motion & Gravitation, and Waves & Sound |
| **Formula Reference** | Quick-glance grid of every major Physics 1 equation |
| **Practice Quiz** | 10 multiple-choice questions with instant feedback and a final score |
| **Resources** | Curated links to Khan Academy, OpenStax, PhET simulations, and The Physics Classroom |

## File Structure

```
index.html       ← main page
css/style.css    ← all styles
js/script.js     ← accordion & quiz logic
```

## How to Run

Simply open `index.html` in a browser:

```bash
open index.html       # macOS
start index.html      # Windows
xdg-open index.html   # Linux
```

Or serve it with any static file server:

```bash
npx serve .
```
